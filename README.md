# thundermail-deploy

Here lie Kustomize manifests for deploying Stalwart-as-Thundermail in Kubernetes.

This project is installed into Kubernetes clusters at Thundermail through [Pulumi and ArgoCD projects](https://github.com/thunderbird/platform-infrastructure/).
